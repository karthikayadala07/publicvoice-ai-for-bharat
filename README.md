# 🏛️ SchemeFinder - Complete Website

## Files in this project:
- `index.html` - Main HTML file (all pages)
- `styles.css` - Complete stylesheet
- `data.js` - Schemes data, career data, quiz questions
- `translations.js` - 7-language translation system
- `app.js` - All JavaScript functionality

## How to Run in VS Code:
1. Open the folder in VS Code
2. Install the "Live Server" extension (by Ritwick Dey)
3. Right-click on `index.html` → "Open with Live Server"
4. Website opens at localhost:5500

## Features:
✅ Login & Register pages
✅ Home page with hero, how-it-works, popular schemes
✅ Scheme finder with filters (age, income, occupation, category, gender, state)
✅ Voice search (tap to speak)
✅ 7 Languages (English, Hindi, Telugu, Tamil, Kannada, Marathi, Bengali)
✅ 12+ government schemes with full details
✅ Step-by-step application guidance
✅ Career Guide (after 10th, 12th, Graduation, Post-Graduation)
✅ AI Career Predictor quiz (8 questions)
✅ FAQs section
✅ AI Chatbot (Sahayak AI) for complaints & support
✅ Text-to-Speech (Listen to scheme details)
✅ Toast notifications
✅ Fully responsive (mobile-friendly)
